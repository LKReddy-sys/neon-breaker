# Neon Breaker

Neon Breaker is a retro-style brick breaker game built with **React** and **Vite**.

## Features

- Neon-styled brick breaker gameplay
- Score, lives, and level display
- Pause / resume with keyboard
- Top navbar with links to my other tools (QR Generator, BMI Calculator, eGFR Calculator, Candy Match)
- Responsive layout for desktop

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind (via CDN)
- Lucide Icons
